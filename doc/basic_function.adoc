= Basic function

Блок обеспечивает передачу транзакций от одного из мастер-устройств к ведомому интерфейсу, осуществляя выбор активного источника на основе сигнала арбитража. При этом остальные мастеры находятся в состоянии ожидания, пока текущая транзакция не будет завершена.
